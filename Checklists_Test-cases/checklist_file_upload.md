# Checklist for image upload (postimages.org) 

- [ ] The "select images" button opens Windows Explorer 
- [ ] If click "Cancel" in Windows Explorer, the file download is canceled
- [ ] After selecting a valid image, downloading starts automatically
- [ ] png file is loaded
- [ ] jpg file is loading
- [ ] jpeg file is loading
- [ ] webp file is loading
- [ ] multiple files of valid format are loaded
- [ ] loading is not possible if the local file is not selected
- [ ] attempt to upload an .exe file: error "You can't upload files of this type."
- [ ] attempt to upload a .txt file: error "You can't upload files of this type."
- [ ] attempt to upload a .pdf file: error "You can't upload files of this type."
- [ ] file name contains special characters: download successful
- [ ] image larger than 15MB: upload successful
- [ ] After uploading a valid image, a preview is displayed, matches the original
- [ ] After uploading a valid image, links to the image is displayed, the image is accessible
